# US-08 — View Training Schedule

**Role:** Current member
**Goal:** Find the current training schedule for indoor and outdoor ranges
**Benefit:** Know exactly when and where to train each season

## Story

> As a current member, I want to view the training schedule for both the indoor (Eddaskolan) and outdoor (Rävsta) ranges so that I know when I can train each season.

## Acceptance Criteria

- [ ] Both indoor and outdoor schedules are visible on the same page
- [ ] Each session shows: location, day, start time, end time, intended audience
- [ ] Seasonal sessions (outdoor only in May–December) are clearly marked
- [ ] The schedule shows current season by default (no need to manually filter)
- [ ] Location details include address or a map link for new members
- [ ] Accessible and readable on mobile without horizontal scroll
- [ ] Schedule is updated via CMS (no developer needed for seasonal changes)

## Notes

- Indoor: Eddaskolan — Monday 18:00–20:00, Sunday 15:00–17:00
- Outdoor: Rävstvägen 110, Märsta (open May–December, weather permitting)
