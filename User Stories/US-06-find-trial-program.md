# US-06 — Find Trial Program Information

**Role:** Person curious about trying archery (prospective visitor)
**Goal:** Quickly find cost, schedule, and location for the trial program
**Benefit:** Decide whether to attend without having to email the club first

## Story

> As a person curious about trying archery, I want to find the trial program page (Prova på) and immediately see the cost, date, and location so that I can decide whether to attend without needing to contact anyone.

## Acceptance Criteria

- [ ] `/prova-pa/` is reachable from the main navigation within 1 click
- [ ] The page prominently shows: cost (100 kr for 3 sessions), schedule (first Monday of each month), time, and location
- [ ] The page mentions the minimum age (10 years)
- [ ] A clear call-to-action is present: either a contact email button or a "Boka via mail" link
- [ ] The page explains that equipment is provided (no need to bring anything)
- [ ] The page is readable on mobile
- [ ] Page load time is under 2 seconds on a 4G connection

## Notes

- This is one of the most important conversion pages on the site
- CTA should pre-fill email subject line: "Prova på bågskyttning"
