# US-12 — Competition Registration

**Role:** Competition participant (member or external archer)
**Goal:** Find Thors 3D event details and registration information
**Benefit:** Register before the deadline without emailing the club organizer

## Story

> As a competition participant, I want to find the annual Thors 3D event details (date, location, registration link) so that I can register before the deadline.

## Acceptance Criteria

- [ ] Upcoming events are listed on a dedicated page with date, time, and location
- [ ] Each event has a direct link to the registration system (e.g. ianseo, Laget.se)
- [ ] Registration deadline is clearly shown
- [ ] The event describes format (3D forest course, number of targets, scoring method)
- [ ] Pay & Shoot information (150 kr, Swish payment, 24 targets) is present on the competition page
- [ ] Past events are archived (visible but clearly marked as past)
- [ ] Events have JSON-LD `Event` structured data so they can appear in Google's event search

## Notes

- Thors 3D is typically held in May at the Rävsta facility
- Pay & Shoot is available May–December, 150 kr per round
